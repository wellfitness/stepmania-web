🎸 GH AutoStepper

Genera partituras de Guitar Hero (.chart) desde MP3/WAV — listas para Clone Hero o el simulador de Sincro

1 Sube canciones — puedes añadir varias a la cola

2 Ajustes del estilo

⏱ Duración máxima por canción
Recorta el chart Y el audio al tiempo elegido (con fade-out de 1.5s para evitar el corte abrupto). Útil para canciones largas que se hacen pesadas. El audio recortado se exporta como WAV; el original se mantiene si eliges "Completa".
1.7
Más bajo = detecta más notas. Sub-1.0 puede meter ruido en canciones suaves.
18%
Probabilidad de que una nota suene con un color adyacente simultáneo (siempre par de colores). El valor se aplica a Difícil y Experto; Intermedio usa la mitad para tenerlos ocasionales sin saturar. Fácil nunca tiene colores simultáneos.
20%
Probabilidad de que una nota con gap ≥ 1/2 beat al siguiente se convierta en sustain.

3 Generar partituras — se procesa toda la cola

Al pulsar Generar partituras, cada canción se analiza una a una (BPM, onsets, dificultades) y produce un .chart con sus 4 dificultades. Cuando termine, descargarás todo en un ZIP en el paso 4.

4 Descarga

Cada ZIP contiene notes.chart + song.ini + el audio. Para Clone Hero: extrae cada ZIP en una carpeta dentro de /Songs/. Para tocar en Sincro, guarda en biblioteca y abre desde Tocar.
